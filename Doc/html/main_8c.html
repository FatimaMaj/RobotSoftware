INTERRUPT ( Timer3_ISR  ,
INTERRUPT_TIMER3   
)

Interrupt vector for the timer3.

Responsible for changing commissioningState, fires every 33ms since (24*10^6) / 12 = 2Mhz and 1/(2*10^6) = 5*10^-7 -> 50us per/tick 0xFFFF base10-> 65535. Overflow will occure 65535 * 5*10^-7 = 0.0328s
